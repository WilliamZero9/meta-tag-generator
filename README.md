# Meta Tag Generator

A clean, fast web tool that generates copy-paste-ready HTML meta tags for Open Graph, Twitter Cards, and standard SEO.

## Features

- Live preview as you type
- Open Graph, Twitter Card, and standard SEO meta tags
- One-click copy to clipboard
- Mobile responsive, dark theme
- No frameworks, no dependencies -- vanilla HTML/CSS/JS

## Usage

Fill in your page details and copy the generated meta tags into your HTML `<head>`.

## Deployment

Deployed on Netlify free tier. Push to trigger a build, or deploy manually:

```bash
netlify deploy --prod --dir=.
```
